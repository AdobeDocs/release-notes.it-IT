# Note sulla versione di Marketing Cloud

Marketing Cloud.

## Servizi

### Interfaccia Marketing Cloud {#marketing_cloud_interface}

Versione 15.7 release pianificata per il: **22 luglio 2015 **

* È stato risolto un problema che impediva l&#39;aggiornamento nei report di Analytics delle descrizioni dell&#39;attributo specificato nella pagina Visualizza/modifica schema (negli attributi del cliente). (MAC-25985)
* È stato risolto un problema che impediva alle miniature di eseguire il rendering per le risorse caricate. (MAC-25863)
* È stato risolto un problema che impediva ai nuovi segmenti creati in Reports &amp; Analytics di essere disponibili in Audience di Marketing Cloud. (MAC-25817)
* È stato risolto un problema che impediva al pubblico di eseguire condivisioni da Analytics se si utilizzava il servizio ID visitatore. (MAC-25788, MAC-25747)
* È stato aggiunto un supporto per caratteri multibyte negli attributi del cliente. (MAC-25552)

**Problema noto**

Un problema noto crea la duplicazione di account automaticamente generati in Audience Manager e li collega automaticamente a un&#39;identità Marketing Cloud del cliente. Il problema si verifica quando si tenta di spostarsi ad Audience Manager prima di collegare i propri account. Adobe consiglia di collegare gli account Audience Manager a Marketing Cloud prima di spostarsi ad Audience Manager. (MAC-25640)

Per assistenza, consulta la [documentazione del prodotto Marketing Cloud](https://marketing.adobe.com/resources/help/en_US/mcloud/).

## Analytics

### Correzioni di Reports &amp; Analytics {#fixes_reports_analytics}


* È stato risolto un problema che si verificava durante l&#39;utilizzo di un segmento tra più suite per report. (AN-104828)
* È stato modificato il limite del numero di suite per report che possono essere visualizzate in Admin Console quando vengono visualizzate le colonne personalizzare dalla 500 alla 400. (AN-102864)
* È stato risolto un problema con il servizio Content Aware: la dimensione della colonna era aumentato a 255 per consentire l&#39;utilizzo di dati con codifica UTF-8. (AN-103834)
* È stato risolto un problema con il menu dei report Conversione personalizzata che non caricava l&#39;interfaccia utente.(AN-103816)
* È stato risolto un problema che impediva la generazione di report in formato [!DNL  .pdf]. (AN-103037)
* È stato risolto un problema con il grafico in alcuni report poiché mostrava dati di un mese futuro. (AN-104372 e AN-101257)
* È stato risolto un problema con Mostra segmenti che impediva il caricamento nei report del dashboard. (AN-105088)
* È stato risolto un problema con i dashboard che impediva la visualizzazione di elementi leggibili. Gli elementi erano troncati nel mezzo di un carattere multibyte. (AN-104940)
* È stato risolto un problema con i calendari personalizzati che impediva la restituzione di dati corretti. (AN-103825)
* È stato risolto un problema relativo al mancato funzionamento del download di PDF per le metriche utilizzando il formato ora. (AN-101515)
* È stato risolto un errore che causava la restituzione di dati non corretti dalle metriche calcolate contenenti segmenti. (AN-105320)

### Correzioni di Report Builder {#fixes_arb}

* È stato risolto un problema che impediva la visualizzazione della metrica calcolata Minuti totali nell&#39;interfaccia utente. (AN-105218)

### Correzioni di Ad Hoc Analysis {#fixes_ad_hoc}

* È stato risolto un problema con i dati mancanti relativi alle Città nel report Città (GeoSegmentation). (AN-104987)
* È stato risolto un problema che impediva alle metriche di conversione di caricarsi propriamente. (AN-104957)
* È stato risolto un problema che si verificava usando più di quattro operazioni matematiche durante la creazione di una metrica calcolata e che dava come risultato un errore. (AN-105362)

### Correzioni di Data Warehouse {#fixes_data_warehouse}

* È stato corretto il supporto variabili/intervallo data per i segmenti. (AN-105271)
* È stato risolto un problema che impediva la generazione di file tableau con caratteri UTF-8. (AN-75471)
* È stata corretta una deduplicazione della metrica evento del contatore nei report con più prodotti per hit e suddivisione nella classificazione del prodotto.
* I risultati dei report dalla metrica Visualizzazioni pagina in Data Warehouse ora sono completamente allineati con i risultati in Reports &amp; Analytics. (AN-97619)
