## [!DNL Experience Cloud] e [!UICONTROL Servizi di base]

Note sulla versione dell’interfaccia dei servizi core, compresi [!UICONTROL Assets], [!UICONTROL Feed], [!UICONTROL Notifiche], Persone, [!UICONTROL Mobile] Services, Launch e Dynamic Tag Management. Sono incluse anche le informazioni sulla versione del servizio Experience Cloud ID (già noto in Analytics come *`Visitor ID service`* o *`Visitor API`* e *`Marketing Cloud ID service`*).

### [!DNL Experience Cloud] Interfaccia {#experience_cloud_interface}

Nuove funzioni e correzioni nell’interfaccia di [!DNL  Adobe Experience Cloud].

**Correzioni**

* È stato risolto un problema per il quale la gerarchia di directory del server Adobe Marketing Cloud e il layout del file system erano visibili tramite  Adobe Campaign. Sono stati apportati miglioramenti per nascondere la gerarchia delle directory e le cartelle limitate. (CORE-5963)
* È stato risolto un problema che impediva l&#39;impostazione di una lingua di notifica preferita predefinita per gli utenti  Experience Cloud. (CORE-9221)
* È stato risolto un problema per il quale i segmenti condivisi da Libreria Pubblico a  Adobe Campaign non erano visibili se il codice della lingua non era impostato.
* È stato aggiornato l&#39;URL di scambio  Adobe (https://marketing.adobe.com/resources/content/resources/en/exchange/marketplace.html) in modo che venga reindirizzato al nuovo URL del Experience Cloud  (https://experiencecloud.adobeexchange.com). (CORE-10068)
See [Experience Cloud Product Documentation](https://docs.adobe.com/content/help/it-IT/core-services/interface/about-core-services/core-services-landing.html) for product help.
